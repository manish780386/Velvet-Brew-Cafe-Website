# ☕ Velvet Brew Café Website

A modern **Full Stack Café Management Website** built using **React, Django, SQL, HTML, CSS, and JavaScript**.
This project allows customers to view menu items, place orders, book tables, and contact the café, while admins can manage everything from a secure dashboard.

---

## 🚀 Features

### 👨‍🍳 Customer Side

* Beautiful Home Page with animations
* Coffee Menu with Add-to-Cart system
* Table Booking system
* Contact Form
* Animated Gallery
* Responsive design for mobile & desktop

### 🛠️ Admin Dashboard

* Admin Login
* Manage Menu Items
* View Menu Orders
* View Table Bookings
* View Contact Messages
* Logout system

## ⚙️ Installation Steps

### 1️⃣ Clone Project

```bash
git clone <respo-link>
cd velvet-brew-cafe
```

---

### 2️⃣ Frontend Setup

```bash
cd frontend
npm install
npm start
```

---

### 3️⃣ Backend Setup

```bash
cd backend
python -m venv venv
venv\Scripts\activate
python manage.py migrate
python manage.py runserver
```

---

### 4️⃣ Admin Login

Create admin user:

```bash
python manage.py createsuperuser
```
